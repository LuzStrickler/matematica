FUNCIONES EXPONENCIAL

Fue obra del matemático suizo, Leonhard Euler.

¿QUE ES UNA FUNCIÓN EXPONENCIAL?

Es un tipo de función matemática en la que la variable independiente se encuentra en el exponente de una base constante y se representa con la fórmula: f(x) = k.(a ^ (x + b)) + c, donde:


  • - "f(x)" es el valor de la función en un punto dado "x".

  • - "k" tiene que ser distinto de 0 y pertenecer al conjunto de números reales.

  • - "a" tiene que ser mayor a 0 y distinto de 1.


ANÁLISIS DE PARAMETROS

El análisis de los parámetros en una función exponencial implica comprender el significado de "a", "b", "c" y "k".


  • - "a": Cuando "a" es mayor a 1 la función será creciente. Si "a" esta entre 0 y 1 la función será decreciente.


  • - "b": Cuando "b" es mayor a 0 la función se desplaza hacia la izquierda. Si "b" es menor a 0 la función se desplaza a la derecha.


  • - "c": Cuando "c" es mayor a 0 la función se desplaza hacia arriba. Si "c" es menor a 0 la función se desplaza hacia abajo.


  • - "k": Cuando "k" es mayor a 0 la función crece más rápido. Si "k" es menor a 0 la función decrece y se invierte el movimiento de la parábola.
